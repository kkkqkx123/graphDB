# Inversearch 服务功能分析

## 概述

Inversearch 是一个基于 Rust 实现的高性能关键词搜索服务，基于 FlexSearch 核心逻辑。它提供了全文搜索、倒排索引、结果高亮等功能，支持 gRPC 协议通信。

## 核心功能模块

### 1. 索引模块 (src/index/)

**功能描述**：
- 实现高性能倒排索引
- 支持多种分词模式（严格、正向、反向、全量、双向）
- 支持上下文感知索引
- 可选快速更新模式

**核心结构**：
```rust
pub struct Index {
    pub map: KeystoreMap<String, Vec<DocId>>,     // 主索引映射
    pub ctx: KeystoreMap<String, Vec<DocId>>,     // 上下文索引
    pub reg: Register,                             // 文档注册表
    pub resolution: usize,                         // 分辨率
    pub tokenize: TokenizeMode,                    // 分词模式
    pub depth: usize,                              // 深度
    pub bidirectional: bool,                       // 双向索引
    pub fastupdate: bool,                          // 快速更新
    pub encoder: Encoder,                          // 编码器
    pub cache: Option<SearchCache>,                // 搜索缓存
}
```

**主要方法**：
- `add()` - 添加文档到索引
- `remove()` - 从索引移除文档
- `update()` - 更新文档
- `search()` - 执行搜索
- `search_cached()` - 带缓存的搜索
- `clear()` - 清空索引

```rust
    store: Option<HashMap<DocId, Value>>, // 文档存储

}
```

**子模块**：
- `field.rs` - 字段定义和配置
- `tree.rs` - 树形结构解析

## gRPC 服务接口

**服务定义** (`proto/inversearch.proto`)：

```protobuf
service InversearchService {
  rpc AddDocument(AddDocumentRequest) returns (AddDocumentResponse);
  rpc UpdateDocument(UpdateDocumentRequest) returns (UpdateDocumentResponse);
  rpc RemoveDocument(RemoveDocumentRequest) returns (RemoveDocumentResponse);
  rpc Search(SearchRequest) returns (SearchResponse);
  rpc ClearIndex(ClearIndexRequest) returns (ClearIndexResponse);
  rpc GetStats(GetStatsRequest) returns (GetStatsResponse);
}
```

**主要操作**：
- `AddDocument` - 添加文档
- `UpdateDocument` - 更新文档
- `RemoveDocument` - 删除文档
- `Search` - 搜索（支持上下文、建议、解析、富化、缓存选项）
- `ClearIndex` - 清空索引
- `GetStats` - 获取统计信息
