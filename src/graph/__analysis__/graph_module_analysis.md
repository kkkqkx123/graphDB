# src/graph 目录功能分析

src/graph 目录是图数据库的核心操作模块，主要负责以下功能：

## 1. 图数据模式定义 (schema.rs)
- 定义图数据库的模式结构，包括顶点类型、边类型和索引
- 定义属性的数据类型和约束
- 提供索引定义和实体类型枚举

## 2. 事务管理 (transaction.rs)
- 实现图数据库的事务处理机制
- 支持ACID特性，提供begin、commit、rollback等操作
- 管理事务状态和操作队列

## 3. 查询结果处理 (result_set.rs)
- 定义查询结果集的结构
- 包含执行统计信息，如扫描的顶点和边的数量、执行时间等
- 提供添加行和管理结果集的方法

## 4. 响应结构定义 (response.rs)
- 定义图数据库查询响应的数据结构
- 包括各种类型的图数据（顶点、边、路径等）
- 提供API响应格式化功能

## 5. 批量操作 (batch_operation.rs)
- 提供批量插入、更新、删除等操作的支持

## 6. 索引系统 (index.rs)
- 实现图数据库的索引功能，提高查询效率

## 7. 工具函数 (utils/)
- 提供图操作相关的辅助函数

总的来说，src/graph 目录是整个图数据库系统的核心组件，它封装了图数据的基本操作、事务管理、模式定义和查询响应等功能，为上层API和服务提供了统一的图操作接口。
